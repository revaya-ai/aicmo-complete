# Visual Brand — Lumen Skin Studio

The Studio (Station 2) renders posts against this. Brand QC scores against it.

## Colors
- **Background:** warm off-white `#F7F3EC`
- **Accent:** soft terracotta `#C77B58`
- **Ink (text):** deep brown `#2E2620`
- **Muted:** sand `#D9CFC1`

## Fonts
- **Headline:** a clean humanist serif (e.g. "Fraunces", "Playfair Display")
- **Body:** a quiet sans-serif (e.g. "Inter", "Work Sans")

## Layout feel
Calm, airy, lots of whitespace. Editorial, not loud. Looks like a small
apothecary, not a tech startup. One focal line, plenty of breathing room.

## Voice descriptors (visual)
honest, warm, minimal, tactile, unhurried

## Never (the visual "never" list)
- Never use neon or high-saturation colors.
- Never use stock-photo "perfect skin" close-ups.
- Never use more than two fonts in one post.
- Never use clip-art icons or emoji-heavy graphics.
- Never crowd the layout. White space is the brand.
- Never use grey as a primary color (use sand `#D9CFC1`).
